Internal CSS dan Eksternal CSS

Internal CSS

Kode CSS internal diletakkan di dalam bagian head pada halaman. Class dan ID bisa digunakan untuk merujuk pada kode CSS, namun hanya akan aktif pada halaman tersebut.

Eksternal CSS

File CSS eksternal biasanya diletakkan setelah bagian head pada halaman HTML. Dengan cara menghubungkannya ke file .CSS eksternal sehingga file HTML dan CSS terpisah

Inline CSS

Inline CSS diletakkan didalam tag HTML tertentu yang ingin diberi style. Namun cara ini kurang direkomendasikan, karena setiap tag HTML harus diberikan style satu per satu
